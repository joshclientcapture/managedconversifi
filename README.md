# Conversifi Dashboard

Managed customer dashboard for Conversifi clients.

## Overview

This dashboard provides Conversifi's managed clients with access to:
- Campaign performance statistics
- Booking management
- Weekly reports

## Technologies

- React + Vite
- TypeScript
- Tailwind CSS
- Shadcn/ui Components

## Development

```sh
npm install
npm run dev
```

## Deployment

The application is deployed and accessible via the published URL.
